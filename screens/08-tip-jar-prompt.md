# Screen 08 - Tip jar prompt (small, dismissible)

**Purpose:** a friendly, optional nudge to tip.
**When:** appears ONCE after a happy moment (e.g. right after marking a test a 5-star keeper, or
after a useful AI suggestion), then never again.

**Shows:** "Enjoying Kilnkeep?" + a short line + a "Buy me a coffee" button + a "Not now" button + an X to close.

**Rules:** never naggy. Show once, easy to dismiss. Remember that it was dismissed (store a flag) so it doesn't come back.
