# Screen 06 - Firings

**Purpose:** your kiln firings.
**Reached:** the Firings tab.

**Shows (list):** firing cards - date, kiln, type (bisque / glaze), target cone, number of tests in that firing.

**Shows (new / edit):** kiln, date, type, target cone, atmosphere (oxidation / reduction), an
optional schedule (ramp rows: rate / temp / hold), optional cost, notes.

**Buttons / actions**
- "+ New firing" -> create a firing.
- "Save".
- Tap a firing -> see the tests fired in it.

**Data:** reads the user's `firings`.
